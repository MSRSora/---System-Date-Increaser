---系统日期增加器 - System Date Increaser
用于快速增加Windows系统日期，可用于快速刷取含挂机要素的单机游戏的奖励，而无需频繁调整系统日期。
This tool is used to quickly update the date in the Windows system. It can also be used to quickly obtain the rewards of single-player games that contain cheat elements, without the need to frequently adjust the system date.
